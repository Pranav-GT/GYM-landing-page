A clean, fully responsive gym landing page built with HTML, CSS, and JavaScript. The page includes a hero section with key stats (105+ trainers, 970+ members, 135+ programs), four fitness program cards (Strength Training, Basic Yoga, Body Building, Weight Loss), a services highlights section, a Swiper.js-powered testimonials slider, and a newsletter-ready footer. Scroll animations are handled via ScrollReveal, and the nav collapses into a hamburger menu on mobile.
